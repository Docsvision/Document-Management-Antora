= Работа с  заданиями и группами заданий

В приложении _{dm}_ можно создавать различные варианты заданий для исполнителей, как индивидуальные, так и групповые.

Для индивидуальных заданий предназначена специальная карточка xref:TC_Descr.html[_Задание УД_]. Данная карточка может быть создана различными xref:task_Task_Creation.adoc[способами].

Для групп заданий предназначена специальная карточка xref:GroupTasks_Card.html[_Группа заданий УД_]. Данная карточка также может быть создана различными xref:task_GroupTask_create.adoc[способами].
