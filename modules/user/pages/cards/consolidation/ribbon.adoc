= Лента карточки "Консолидация договора"

.Кнопки ленты инструментов карточки задания
[cols=",,,",options="header"]
|===
|Группа |Кнопка |Название |Назначение

|Общие

|image:buttons/save.png[Дискета]
|Сохранить.
|Сохраняет сделанные в карточке изменения.

.4+|Действия

|image:buttons/to-work.png[Лист с иконкой треугольника]
|В работу.
|Переводит карточку задания в состояние `В работе`.

|image:buttons/delegate-contract.png[Лист с иконкой сотрудника]
|Делегировать.
|Открывает окно _Делегирование_ для делегирования задания.

|image:buttons/open-contract.png[Лист с загнутым уголком]
|Открыть карточку документа.
|Открывает карточку согласуемого документа (договора или дополнительного соглашения).

|image:buttons/letter-send.png[Письмо со стрелкой]
|Отправить письмо контрагенту.
|Формирует письмо контрагенту с вложенной итоговой версией документа.

.3+|Варианты завершения

|image:buttons/pen.png[Ручка]
|На подписание.
a|Закрывает карточку задания и переводит договорной документ в состояние `На подписании`.

include::partial$excerpts.adoc[tags=on-stage]

|image:buttons/partner-sign.png[Человек и ручка]
|На согласование с контрагентом.
a|Закрывает карточку задания и переводит договорной документ в состояние `На согласовании с контрагентом`.

Кнопка доступна консолидатору на этапе консолидации согласования или подписания.

|image:buttons/cancel.png[Перечёркнутый круг]
|Отменить.
a|Открывает окно _Параметры завершения задания_ для ввода комментария о причинах отмены маршрута согласования. Переводит договорной документ в состояние `На подготовке`.
