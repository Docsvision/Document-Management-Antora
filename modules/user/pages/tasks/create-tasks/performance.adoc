= Задание на исполнение

Задание вида _На исполнение_ предназначено для раздачи исполнителям поручений, связанных с приложенным документом.

Создать задание на исполнение из карточки _Документ УД_ можно несколькими способами.

. Создайте карточку _Задание УД_ вида _На исполнение_ любым из следующих способов:
* <<ribbon,С помощью специальной кнопки на панели инструментов карточки>>.
* <<context,С помощью контекстного меню вкладки "Задания">>.
+
Также для данного вида в _Справочнике видов карточек_ по умолчанию настроен режим создания _Задание без размещения_, позволяющий создавать задания с использованием команды на ленте {wincl}а *Создать задание*, см. xref:tasks/create-tasks/from-ribbon.adoc[Создание заданий из представления {wincl}а].
+
В результате выполненных действий будет открыта карточка задания на исполнение.
+
.Вид задания на исполнение
image::task-performance.png[Вид задания на исполнение]
+
. Заполните поля карточки.
.. В поле _Название_ укажите название задания.
.. В поле _Автор_ выберите автора задания или укажите его заместителя. Для этого нажмите на кнопку image:buttons/three-dots.png[Три точки] и выберите запись из _Справочника сотрудников_. Если запись добавлена ошибочно, удалите ее кнопкой image:buttons/x-black.png[Крестик].
+
По умолчанию в поле указывается сотрудник, создавший карточку входящего документа.
+
.. В поле _Исполнители_ xref:tasks/create-tasks/select-performer.adoc[выберите] исполнителя для задания.
.. При необходимости настройте xref:tasks/create-tasks/deputy.adoc[замещение].
.. Выберите важность для задания.
.. В поле _Дата завершения_ укажите планируемую дату завершения задания. Для этого нажмите на кнопку image:buttons/triangle-down.png[Треугольник вниз] и выберите дату из встроенного календаря.
.. При необходимости xref:tasks/create-tasks/control.adoc[настройте] контроль за выполнением задания.
.. При необходимости xref:task_Task_Approve.adoc[настройте] дополнительную приёмку задания _Контролером_.
.. Настройте xref:tasks/create-tasks/reminders.adoc[напоминания].
. Если требуется, приложите файлы и ссылки к заданию на странице _Файлы и ссылки_.
+

****
+
. Для отправки задания исполнителю нажмите на кнопку *Отправить* в карточке задания.
+
При этом карточка задания у автора будет закрыта. Исполнитель получит доступ к данной карточке в папке menu:Мои задания[Входящие].

[#ribbon]
== Создание задания с помощью команды ленты карточки документа

Нажмите на кнопку *Задание* на ленте карточки, чтобы создать новое задание.

.Расположение кнопки "Задание"
image::Task_Creafe_Perform.png[Расположение кнопки "Задание"]

Будет открыта карточка создаваемого задания.

[NOTE]
====
Вид создаваемого задания определяется администратором {dv} (см. "xref:admin:card-kinds/related-tasks.adoc[]").
====

[#context]
== Создание задания из контекстного меню вкладки "Задания"

Чтобы создать задание на исполнение при помощи кнопки на ленте инструментов карточки _Документ_, выполните следующие действия:

В карточке _Документ УД_ перейдите на вкладку _Задания_ и откройте контекстное меню в области списка заданий.

.Контекстное меню в области списка заданий
image::task-create-context.png[Контекстное меню в области списка заданий]

.Окно "Справочник видов карточек"
image::kinds-dir.png[Окно "Справочник видов карточек"]

В результате выполненных действий будет открыта карточка задания на исполнение.
