= Выбор варианта исполнения заданий

Для карточки _Группа заданий УД_ предусмотрены два варианта исполнения заданий: последовательный и параллельный.

При выборе варианта *_Последовательный_* исполнители получат задания для каждого из указанных в карточке исполнителей в той же очередности, в которой они указаны в списке исполнителей.

При выборе варианта *_Параллельный_* все исполнители получат задания одновременно.
