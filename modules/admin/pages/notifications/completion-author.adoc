= Уведомление Автора о завершении задания

.По умолчанию отправка данного уведомления настроена для видов карточек:
* xref:notifications/completion-peformance.adoc["Задание УД -- на исполнение"].
* xref:notifications/completion-acknowledgement.adoc["Задание УД -- на ознакомление"].
* xref:notifications/completion-approval.adoc["Задание КС -- на согласование"].
* xref:notifications/completion-approval.adoc["Задание КС -- на подписание"].
* xref:notifications/completion-approval.adoc["Задание КС -- на консолидацию"].

При необходимости отправка уведомлений может быть настроена для карточек других видов.
