<.> `MessageType`:
+
--
* `0` -- уведомление о завершении задания / группы заданий.
* `1` -- уведомление об отзыве задания.
* `2` -- уведомление о поступление на приёмку.
--
+
--
`Description` -- заголовок сообщения.
--
+
<.> `SendInfo`:
+
* `Date` -- дата.
* `CardUrl` -- ссылка для открытия карточки в клиенте {dv}.
+
<.> `Data` -- xml карточки.
